# TM_CNN
Мы работаем в Почте и хотим автоматизировать сортировку писем по индексу. Клиенты пишут индекс от руки, и нам нужно научиться распознавать эти цифры. У нас есть знаменитый датасет MNIST — 60 000 рукописных цифр для обучения и 10 000 для проверки. Ваша задача — построить модель, которая будет определять цифру по картинке. Начните с простого многослойного перцептрона, а затем попробуйте сверточную сеть. Сравните результаты и сделайте вывод, какой подход лучше для реального применения.
